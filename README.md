目標:

遠隔の通信に関して学習し、以下の要件を満たすプログラムを制作する							

 ・NAT超えによる遠隔通信を可能とする						

 ・イベント単位での同期を行う						

 ・2人以上との通信を想定する						
							
							
							
想定している形式							

 ・24年冬で作成した描画機構を流用した見下ろし2Dアクションゲーム						

 ・移動、近接攻撃アクションなどの単位で同期						

 ・1人がサーバープレイヤーとなる						

 ・可能ならロールバックができるように設計						
							
							
評価基準							

 ①NAT超えを行える						

 ②イベント同期が行える						

 ③1v1で通信(同拠点内)						

 ④3人以上での対戦						

 ⑤ロールバック対応
